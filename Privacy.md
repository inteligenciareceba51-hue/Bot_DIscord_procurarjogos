Política de Privacidade
Última atualização: 12 de Junho de 2026
Esta Política de Privacidade descreve como o nosso bot (doravante referido como "o Bot") coleta, utiliza e protege as informações dos usuários no Discord. O nosso objetivo primordial é fornecer informações sobre jogos gratuitos de forma transparente e segura, sem comprometer a sua privacidade.
1. Coleta de Dados
O Bot foi desenvolvido visando a máxima privacidade dos usuários. Nós não coletamos, armazenamos ou compartilhamos nenhum dado pessoal identificável.
Durante o seu funcionamento normal, o Bot interage com o Discord apenas para ler comandos (como !checar) e enviar mensagens no servidor. O Bot não guarda registros (logs) permanentes de quem usou o comando, qual servidor foi utilizado ou qualquer outra informação do seu perfil.
2. Uso de Dados
O Bot solicita a intenção de "Message Content" (Conteúdo da Mensagem) exclusivamente para identificar os comandos digitados nos canais onde ele está presente. O conteúdo das mensagens lidas é processado momentaneamente apenas para verificar se corresponde a um comando válido e é imediatamente descartado. Nenhuma mensagem é salva em bancos de dados ou arquivos.
3. Dados de Terceiros
Para fornecer a lista de jogos gratuitos, o Bot faz requisições a uma API externa pública (GamerPower). Nenhuma informação dos usuários do servidor ou do Discord é enviada para essa ou qualquer outra API externa durante esse processo.
4. Retenção de Dados
Uma vez que o Bot não possui um banco de dados, a taxa de retenção de dados dos usuários é estritamente zero.
5. Contato e Suporte
Se você tiver alguma dúvida sobre esta Política de Privacidade ou sobre o funcionamento do Bot, sinta-se livre para entrar em contato com o desenvolvedor responsável através do próprio Discord ou removendo o Bot do seu servidor caso não concorde com estes termos.
